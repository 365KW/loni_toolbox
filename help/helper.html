取色板

📌 取色板

获取颜色值，支持多种颜色代码格式转换。

功能说明

  • 颜色选择器：打开系统颜色选择器对话框选择颜色
  • 格式转换：支持将颜色代码以为多种格式输出

支持的格式

  • HEX - 十六进制颜色代码（如 #FF0000）
  • RGB - 红绿蓝三原色（如 rgb(255, 0, 0)）
  • HSL - 色相、饱和度、亮度
  • HSV - 色相、饱和度、明度
  • HWB - 色相、白度、黑度
  • NCol - 自然颜色系统
  • CIEXYZ - CIE XYZ 色彩空间
  • CIELAB - CIE LAB 色彩空间
  • Oklab - OKLAB 线性色彩空间
  • Oklch - OKLCH 极坐标色彩空间
  • VEC4 - 四维向量 (RGBA)
  • DEC - 十进制整数
  • HEX 整型 - 十六进制整数

使用方法

  1. 从"格式"下拉菜单选择需要的颜色格式
  2. 点击"颜色选择器"按钮打开颜色选择对话框
  3. 在色板中选择想要的颜色
  4. 点击"确定"后，当前颜色值会显示在标签区域

Image Resizer（图像调整大小）

📌 Image Resizer

调整单张或多张图片的尺寸。

功能说明

  • 单图调整：调整单张图片的尺寸
  • 批量调整：一次性调整多张图片的尺寸

调整模式

  • 填充 (Fill) - 填满目标区域，可能裁剪
  • 适应 (Fit) - 适应目标区域，保持比例
  • 拉伸 (Stretch) - 拉伸到目标尺寸

使用方法

  1. 设置目标宽度和高度（像素）
  2. 选择调整模式
  3. 点击"调整大小"选择单张图片处理
  4. 或点击"批量调整"选择多张图片和输出目录

进制转换器

📌 进制转换器

在不同进制之间转换数字。

功能说明

支持 2 到 60 进制之间的任意转换。

使用方法

  1. 设置"从进制"（输入数字的进制）
  2. 设置"到进制"（输出数字的进制）
  3. 在"输入"框中输入数字
  4. 点击"转换"按钮，结果显示在"结果"框中
常用进制：2（二进制）、8（八进制）、10（十进制）、16（十六进制）

时间戳转换

📌 时间戳转换

在时间戳和日期时间格式之间相互转换。

功能说明

  • 时间戳转日期：将 Unix 时间戳转换为可读日期时间
  • 日期转时间戳：将日期时间转换为 Unix 时间戳
  • 时区支持：支持 UTC、本地时区及其他常见时区

使用方法

  1. 选择目标时区（UTC、Local 或其他）
  2. 输入时间戳获取对应日期时间，或设置各时间分量（年/月/日/时/分/秒）获取时间戳
  3. 点击"时间戳→日期"或"日期→时间戳"按钮进行转换
  4. 可使用右侧的加减按钮微调各时间分量

Base64解密（Base64 编解码）

📌 Base64解密

对文本进行 Base64 编码和解码。

功能说明

  • 编码：将普通文本转换为 Base64 编码
  • 解码：将 Base64 编码还原为普通文本

使用方法

  1. 在"输入"区域输入要处理的文本
  2. 点击"编码"按钮进行 Base64 编码
  3. 或点击"解码"按钮进行 Base64 解码
  4. 结果显示在"输出"区域

哈希/校验和生成器

📌 哈希/校验和生成器

生成文本或文件的哈希值和校验和。

支持的算法

  • MD5 - 128位哈希值（已不建议用于安全场景）
  • SHA1 - 160位哈希值
  • SHA256 - 256位哈希值（推荐）
  • SHA512 - 512位哈希值
  • CRC32 - 32位循环冗余校验

使用方法

  1. 选择哈希算法
  2. 在文本框输入要哈希的文本，或点击"选择文件"选择文件
  3. 点击"生成哈希"按钮
  4. 哈希值显示在结果框中
注意：MD5 和 SHA1 由于存在碰撞风险，不建议用于安全敏感场景

密码生成器

📌 密码生成器

生成随机强密码，并评估密码强度。

功能说明

  • 自定义长度：密码长度 4-128 位可选
  • 字符类型：可选择包含大写字母、小写字母、数字、符号
  • 排除易混淆字符：可排除 0、O、l、1 等易混淆字符
  • 强度评估：显示密码强度（非常弱/弱/一般/强/非常强）
  • 一键复制：快速将密码复制到剪贴板

使用方法

  1. 设置密码长度（默认 16 位）
  2. 勾选要包含的字符类型
  3. 可选：勾选"排除易混淆字符"
  4. 点击"生成密码"按钮
  5. 密码显示在文本框中，同时显示强度评估
  6. 点击"复制"将密码复制到剪贴板
建议密码长度至少 12 位，并包含所有类型的字符以获得更强的安全性
